# ocr-bye
Stay away from OCR , convert your words into image which can not be recognized easily. 
</br>
再见，OCR！远离OCR，将文字转换成OCR无法轻易识别的图片。

## Features
- 文字单独随机旋转角度
- 干扰线
- 文字设置大小
- 生成图片自适应大小
- 每行字数设置
- 添加橫格綫
